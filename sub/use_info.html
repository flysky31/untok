선불 상담 이용방법

1. 회원가입 후 로그인

2. 페이충전 (최소 20,000원)

3. 070-8019-9009로 전화

4. 안내멘트 종료 후 2번 + 상담사 고유번호 입력

선불 상담 이용요금

1. 상담사 전화연결 직후부터 선불 충전 금액이 차감됩니다.

2. 상담사 전화연결 후 30초당 상담사가 설정한 금액만큼 코인이 차감됩니다.

3. 충전된 코인이 모두 사용되면 자동으로 종료됩니다.

4. 선불 충전 상담 중에는 충전이 되지 않습니다.

060 상담 이용방법

1. 원하는 상담사 프로필에서 상담하기 클릭

2. [후불 상담 바로 연결하기] 클릭

3. 안내 멘트에 따라 상담사 고유번호 입력

4. 전화 연결이 되면 상담이 시작됩니다.

상담번호로 직접 전화를 건 경우 상담사의 고유번호 3자리를 입력해주세요.
상담 가격별 060 상담 전화번호가 다르기 때문에 원하는 상담사의 060 상담 번호를
꼭! 확인해주세요.

060 상담 이용요금

- 전화 연결 후 처음 40초간은 요금이 부과되지 않습니다.

- 40초 이후 상담사가 설정한 30초당 금액(VAT별도)의 이용료가 부과됩니다.

- 이용료는 다음달 전화 요금에 합산되어 청구됩니다.

- 선불충전상담을 이용하실 경우 060 상담보다 최대 36% 할인된 요금으로 이용이 가능합니다.
